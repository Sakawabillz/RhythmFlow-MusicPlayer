# RhythmFlow Music Player 🎵

A modern music player built with **React + Tailwind CSS**.  
It allows users to browse tracks, view song details, and play music with a clean UI.

---

## 🚀 Features
- 🎶 Music library loaded from `data.json`
- 🔍 Search & browse tracks
- 📱 Responsive UI with Tailwind CSS
- 🖼️ Song details & album artwork
- ⚡ Fast development powered by Vite

---
